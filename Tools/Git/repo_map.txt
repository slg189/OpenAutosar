# repo_map.txt — 子库划分清单 (供 init_subrepos.bat 读取)
# 格式:  <相对路径>|<仓库名(不含.git)>|<分组>
# 以 # 开头的行为注释; 空行忽略。
#
# 注意: 与项目相关的子库不在此固定清单中, 由 init_subrepos.bat 按 --project 生成
#       (仓库名 <name>_<PROJECT>, 扁平命名/下划线; 工作区检出到对应目录根):
#         - 项目源码: ASW -> asw_<PROJECT>,  CDD -> cdd_<PROJECT>

#         - 项目库:   ASW_Libs -> asw_libs_<PROJECT>,  CDD_Libs -> cdd_libs_<PROJECT>,
#                     BSW_Libs -> bsw_libs_<PROJECT>
#                     (BSW 配置代码与静态代码一并打包为库, 故库也按项目)
#         - MCAL 库:  MCAL_Libs -> MCAL_<PROJECT> (归 MCAL group, 按项目区分;
#                     针对项目已固定单一芯片, 库目录不含 <Chip> 一级)
